opy That


Check It Out

Copy That allows you to click on any text and copy it to your clipboard.

Ex:

Click on The text Below to copy it

Copy this Text.

Now paste that in a new tab.

Usage

At the very bottom of your

<body>

tag in your HTML, put this code:

<script src="http://copythat.ml/dist/main.js"></script>

Now all you need to do is give your tag that you want to copy, an id attribute, like this:

<p id="copy">Copy This Text With a Click</p>

Node Package Manager Install

npm install copythat

Global Install

npm install copythat -g
